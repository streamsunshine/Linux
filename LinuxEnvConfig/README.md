# LinuxEnvConfig

各个文件夹中存储着与对应程序相关的配置文件，这些配置文件具体功能和需要放置在操作系统的那个位置，参见对应文件夹中的readme.md

一些安装软件的博客我会尽量放在对应的readme文件中，一些软件安装时间有些久远可能找不到相应的链接，可以自行搜索，教程有很多。

## bash

bash是linux基本的shell程序，一般默认的shell都是bash，使用bash是使用linux命令行的基础。

文件夹存储着便于bash操作的一些配置，比如alias。

## tmux

tmux是窗口管理软件，能够将命令界面分成几个窗口，可以在不同窗口进行不同的操作，方便写代码和调试程序。同时当需要在不同主机登录的时候，tmux可以保存先前的窗口，使用tmux attach即可恢复。

## vim 

vim是文本编辑工具，可以使用vim编写程序，如能灵活掌握vim的指令，将能大大提高编码效率，甚至可以抛弃鼠标进行编码操作。这里的配置文件，对vim进行看扩展配置，使其具有自动命令补全等功能，使用起来更加方便。

## zsh

zsh是一种shell程序，相比于bash其功能更加强大，比如使用tab进行命令不全的时候，zsh不需要区分大小写。
